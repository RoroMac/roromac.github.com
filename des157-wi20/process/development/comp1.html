This design has four separate interactive elements. The first is the table of contents that uses on.scroll animation effects to change the appearing title on the right.The first chapter is an image with various interactive elements that brings up various information. The second chapter has users press keys to type in the search bar to trigger the opacity of the following image. The third chapter has a sliding gallery shows users images of the various websites that have provided new information but has been since disasembled. The fourth chapter is a description and images of the proposal. Animations could be included if I have enough time.
